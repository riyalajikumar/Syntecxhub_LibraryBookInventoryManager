# Library Book Inventory Manager

## Project Overview
Library Book Inventory Manager is a Python-based console application developed to manage library books efficiently. The system allows users to add books, search books, issue books, return books, and generate inventory reports. Data is stored permanently using JSON files.

## Features
- Add new books
- Search books by title or author
- Issue books to users
- Return issued books
- View all available books
- Generate inventory reports
- JSON-based data persistence

## Technologies Used
- Python 3
- Object-Oriented Programming (OOP)
- JSON File Handling

## How to Run
1. Install Python 3.
2. Open terminal or command prompt.
3. Navigate to the project folder.
4. Run:

```bash
python library_manager.py
```
